Component ID: Arduino
Component Type: ATmega328P Development Board
Description: The Arduino Pro Mini is a compact, minimalist development board based on the ATmega328P microcontroller. It is designed for space-constrained projects and permanent installations where the size of a standard Arduino Uno is prohibitive. Unlike most Arduino boards, it does not include a built-in USB-to-serial bridge; instead, it features a 6-pin header for programming via an external FTDI adapter. It is available in two versions: a 5V/16MHz model and a 3.3V/8MHz model for low-power applications.

Key Parameters:
- Microcontroller: ATmega328P
- Operating Voltage: 3.3V or 5V (Model dependent)
- Input Voltage (RAW): 3.35V - 12V (3.3V version) or 5V - 12V (5V version)
- Digital I/O Pins: 14 (of which 6 provide PWM output)
- Analog Input Pins: 8 (A0-A7)
- Flash Memory: 32 KB (2 KB used by bootloader)
- SRAM: 2 KB
- EEPROM: 1 KB
- Clock Speed: 8 MHz or 16 MHz
