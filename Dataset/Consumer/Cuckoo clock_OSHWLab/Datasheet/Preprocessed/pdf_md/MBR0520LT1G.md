# Description and Purpose:

As announced recently, ON Semiconductor is now onsemi. As a result of this re -branding effort, onsemi’ s product marking and package labelling will undergo changes over time. Some impacted areas will be:

• Product marking that can accommodate onsemi due to package body size, will include onsemi marking • Product marking that is too small to accommodate onsemi will continue to include the ON logo (no change) • Labelling on packaging will also be changed to onsemi

# 説明および目的：

最近発表されたとおり、オン・セミコンダクターは、現在はオンセミとなっています。ブランド再構築の結果として、オンセミの製品マーキングやパッケージラベルは時間の経過とともに変更される可能性があります。影響を受ける分野：

- パッケージのボディサイズによりオンセミ対応が可能な製品マーキングには、オンセミマークを- オンセミが入らない大きさの製品マーキングには、引き続きONロゴを表示（変更なし)- また、梱包のラベルもオンセミに変更されます
