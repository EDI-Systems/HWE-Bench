Component ID: ATtiny85-20SU
Component Type: 8-bit AVR Microcontroller
Description: The ATtiny85 is a low-power CMOS 8-bit microcontroller based on the AVR enhanced RISC architecture. By executing powerful instructions in a single clock cycle, the device achieves throughputs approaching 1 MIPS per MHz, allowing the system designer to optimize power consumption versus processing speed. It features 8KB of In-System Programmable Flash memory, 512B EEPROM, and 512B SRAM. Key peripherals include an 8-bit Timer/Counter with PWM, a high-speed 8-bit Timer/Counter with PWM and dead-time generator, a Universal Serial Interface (USI), and a 4-channel 10-bit ADC. It supports multiple sleep modes and offers six programmable I/O lines.

Key Parameters:
- Flash Memory 8 KB
- EEPROM 512 Bytes
- SRAM 512 Bytes
- Speed Grade 20 MHz
- Operating Voltage 2.7V to 5.5V
- ADC 4-channel 10-bit
- I/O Pins 6
- Operating Temperature -40 to +85 Celsius
