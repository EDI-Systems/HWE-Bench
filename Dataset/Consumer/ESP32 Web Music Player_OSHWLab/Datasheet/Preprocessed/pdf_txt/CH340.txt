Component ID: CH340
Component Type: USB to UART Bridge / USB Serial Interface IC
Description:
The CH340 is a USB bus adapter chip that converts USB to a serial UART interface or a parallel printer port. In its primary serial port mode, it emulates a standard COM port with full hardware flow control signals (RTS, DTR, DCD, RI, DSR, CTS), enabling the upgrade of legacy serial devices to USB or adding virtual serial ports to a computer. It features built-in firmware, supports a wide range of baud rates, and can interface with RS232, RS485, or RS422 standards using external level shifters. Several variants include an integrated clock source, eliminating the need for an external crystal.

Key Parameters:
- USB Specification: Full-speed, compatible with USB 2.0
- Supply Voltage: 5V and 3.3V operation
- Baud Rate Range: 50 bps to 2 Mbps
- Integrated Clock: Available on CH340C/N/K/E/X/B variants (no external crystal required)
- Built-in EEPROM: Available on CH340B variant (for serial number configuration)
- Package Options: SOP-16, SOP-8, SSOP-20, ESSOP-10, MSOP-10
- Flow Control Signals: Supports RTS, DTR, DCD, RI, DSR, CTS
- Infrared Support: CH340R variant supports IrDA SIR (2400 bps to 115200 bps)
