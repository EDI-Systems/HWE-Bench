 Component ID: ILI
Component Type: Color TFT LCD Display Module with Resistive Touchscreen and Integrated Controller
Description:
This is a 2.4-inch diagonal color TFT LCD module with an integrated controller, RAM buffer, and a 4-LED white backlight. It features a resistive touchscreen overlay and supports two primary interface modes: an 8-bit parallel interface for high-speed operation and a 4/5-wire SPI interface for reduced pin count. The breakout board includes level-shifting circuitry for 3-5V logic compatibility, an onboard 3.3V regulator, and connections for an optional microSD card. It is designed to offload graphics processing from the host microcontroller.

Key Parameters:
- Display Resolution: 240 x 320 pixels
- Display Technology: TFT with individual RGB pixel control
- Interface Modes: 8-bit parallel, SPI (4/5-wire)
- Logic Voltage Compatibility: 3V to 5V (with onboard level shifting)
- Backlight: 4 white LEDs, PWM controllable
- Touchscreen Type: 4-wire Resistive (separate Y+, X+, Y-, X- pins)
- Supply Voltage (Vin): 3VDC to 5VDC
- Onboard Output Voltage: 3.3V (regulated)
- SD Card Interface: SPI-compatible (shared bus with TFT in SPI mode)
