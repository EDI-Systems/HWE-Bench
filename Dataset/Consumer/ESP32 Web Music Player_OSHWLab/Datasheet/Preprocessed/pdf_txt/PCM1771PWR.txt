Component ID: PCM1771
Component Type: Micro SD (TF) Card Connector
Description: This is a standard Micro SD (TransFlash) card receptacle used to provide removable, non-volatile data storage for embedded systems. It serves as the bridge between a Micro SD card and a host controller (like a microcontroller or SoC). This specific model supports both the high-performance SDIO (Secure Digital Input Output) protocol and the simpler SPI (Serial Peripheral Interface), making it compatible with everything from high-end processors to basic 8-bit MCUs. It also features a mechanical Card Detect (CDN) pin to allow the system to sense when a card is physically present.


Key Parameters:
* Supported Media: Micro SD / TF Card
* Interface Protocols: SDIO (1-bit/4-bit) and SPI
* Operating Voltage ($V_{DD}$): 2.7V to 3.6V (Typically 3.3V)
* Mechanical Features: Integrated metal shield with multiple ground/mounting points (G1-G4)
