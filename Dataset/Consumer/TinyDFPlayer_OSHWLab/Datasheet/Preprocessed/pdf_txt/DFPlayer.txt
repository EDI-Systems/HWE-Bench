Component ID: DF 
Component Type: MP3 Audio Decoder Module  
Description:  
The DFPlayer Mini is a compact, low-cost MP3 module capable of decoding audio files (MP3, WAV, WMA) from storage media (TF card, U-disk, or internal NOR FLASH) and outputting audio via a built-in 24-bit DAC. It supports multiple control modes (serial, I/O, AD button) and features folder-based file management, adjustable volume/EQ, and advertising interruption functionality, making it suitable for embedded voice prompt and playback applications.

Key Parameters:
- Supported Audio Formats: MP3, WAV, WMA
- Supported Sampling Rates: 8, 11.025, 12, 16, 22.05, 24, 32, 44.1, 48 kHz
- DAC Resolution: 24-bit
- Dynamic Range: 90 dB
- Signal-to-Noise Ratio (SNR): 85 dB
- Storage Support: FAT16/FAT32 file system, up to 32 GB TF card, 32 GB U-disk, 64 MB NOR FLASH
- Folder/Song Capacity: Up to 100 folders, each with up to 255 songs
- Volume Control: 30 levels
- Equalizer (EQ) Modes: 6 levels
