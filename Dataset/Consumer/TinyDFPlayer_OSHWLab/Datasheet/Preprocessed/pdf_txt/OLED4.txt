Component ID: OLED
Component Type: OLED Display Controller/Driver
Description: This is a monochrome passive matrix OLED display module with a 0.96-inch diagonal, 128x64 pixel resolution, and an integrated SSD1306BZ controller/driver IC. It supports multiple host interfaces including I2C, SPI (3-wire and 4-wire), and parallel (6800/8080) via hardware configuration pins. The module includes an internal DC/DC charge pump for power supply generation.
