ESP32-WROVER-E йҮҮз”Ё PCB жқҝиҪҪеӨ©зәҝпјҢESP32-WROVER-IE йҮҮз”ЁиҝһжҺҘеҷЁиҝһжҺҘеӨ–йғЁеӨ©зәҝгҖӮдёӨж¬ҫжЁЎз»„еқҮй…ҚзҪ®дәҶ 4MB SPI flash е’Ң 8 MB SPI PSRAMгҖӮжң¬ж–ҮжЎЈжҸҗдҫӣзҡ„дҝЎжҒҜйҖӮз”ЁдәҺиҝҷдёӨж¬ҫжЁЎз»„гҖӮ

# иҜҙжҳҺпјҡ

ESP32-WROVER-E е’Ң ESP32-WROVER-IE йҮҮз”Ёзҡ„иҠҜзүҮжҳҜ ESP32 зі»еҲ—зҡ„ ESP32-D0WD-V3 \*гҖӮ

ESP32-D0WD-V3 иҠҜзүҮе…·жңүеҸҜжү©еұ•гҖҒиҮӘйҖӮеә”зҡ„зү№зӮ№гҖӮдёӨдёӘ CPU ж ёеҸҜд»Ҙиў«еҚ•зӢ¬жҺ§еҲ¶гҖӮCPU ж—¶й’ҹйў‘зҺҮзҡ„и°ғиҠӮиҢғеӣҙдёә 80 MHz еҲ° 240 MHzгҖӮз”ЁжҲ·еҸҜд»Ҙе…ій—ӯ CPU зҡ„з”өжәҗпјҢеҲ©з”ЁдҪҺеҠҹиҖ—еҚҸеӨ„зҗҶеҷЁзӣ‘жөӢеӨ–и®ҫзҡ„зҠ¶жҖҒеҸҳеҢ–жҲ–жҹҗдәӣжЁЎжӢҹйҮҸжҳҜеҗҰи¶…еҮәйҳҲеҖјгҖӮESP32 иҝҳйӣҶжҲҗдәҶдё°еҜҢзҡ„еӨ–и®ҫпјҢеҢ…жӢ¬з”өе®№ејҸи§Ұж‘ёдј ж„ҹеҷЁгҖҒйңҚе°”дј ж„ҹеҷЁгҖҒSD еҚЎжҺҘеҸЈгҖҒд»ҘеӨӘзҪ‘жҺҘеҸЈгҖҒй«ҳйҖҹ SPIгҖҒUARTгҖҒI2S е’Ң I2C зӯүгҖӮ

# иҜҙжҳҺпјҡ

\* е…ідәҺ ESP32 зі»еҲ—иҠҜзүҮзҡ„дә§е“ҒеһӢеҸ·иҜҙжҳҺиҜ·еҸӮз…§ж–ҮжЎЈ гҖҠESP32 жҠҖжңҜи§„ж јд№ҰгҖӢжЁЎз»„йӣҶжҲҗдәҶдј з»ҹи“қзүҷгҖҒдҪҺеҠҹиҖ—и“қзүҷе’Ң Wi-FiпјҢе…·жңүе№ҝжіӣзҡ„з”ЁйҖ”пјҡWi-Fi ж”ҜжҢҒжһҒеӨ§иҢғеӣҙзҡ„йҖҡдҝЎиҝһжҺҘпјҢд№ҹж”ҜжҢҒйҖҡиҝҮи·Ҝз”ұеҷЁзӣҙжҺҘиҝһжҺҘдә’иҒ”зҪ‘пјӣиҖҢи“қзүҷеҸҜд»Ҙи®©з”ЁжҲ·иҝһжҺҘжүӢжңәжҲ–иҖ…е№ҝж’ӯ BLE Beacon д»ҘдҫҝдәҺдҝЎеҸ·жЈҖжөӢгҖӮESP32 иҠҜзүҮзҡ„зқЎзң з”өжөҒе°ҸдәҺ 5 ВөAпјҢдҪҝе…¶йҖӮз”ЁдәҺз”өжұ дҫӣз”өзҡ„еҸҜз©ҝжҲҙз”өеӯҗи®ҫеӨҮгҖӮжЁЎз»„ж”ҜжҢҒзҡ„ж•°жҚ®дј иҫ“йҖҹзҺҮй«ҳиҫҫ 150 MbpsпјҢеӨ©зәҝиҫ“еҮәеҠҹзҺҮиҫҫеҲ° 20 dBmпјҢеҸҜе®һзҺ°жңҖеӨ§иҢғеӣҙзҡ„ж— зәҝйҖҡдҝЎгҖӮеӣ жӯӨпјҢиҝҷж¬ҫжЁЎз»„е…·жңүиЎҢдёҡйўҶе…Ҳзҡ„жҠҖжңҜи§„ж јпјҢеңЁй«ҳйӣҶжҲҗеәҰгҖҒж— зәҝдј иҫ“и·қзҰ»гҖҒеҠҹиҖ—д»ҘеҸҠзҪ‘з»ңиҒ”йҖҡзӯүж–№йқўжҖ§иғҪжһҒдҪігҖӮ

ESP32 зҡ„ж“ҚдҪңзі»з»ҹжҳҜеёҰжңү LwIP зҡ„ freeRTOSпјҢиҝҳеҶ…зҪ®дәҶеёҰжңүзЎ¬д»¶еҠ йҖҹеҠҹиғҪзҡ„ TLS 1.2гҖӮиҠҜзүҮеҗҢж—¶ж”ҜжҢҒ OTA еҠ еҜҶеҚҮзә§пјҢж–№дҫҝз”ЁжҲ·еңЁдә§е“ҒеҸ‘еёғд№ӢеҗҺз»§з»ӯеҚҮзә§гҖӮ

# 4.1 CPU е’ҢеҶ…еӯҳ

ESP32-D0WD-V3 еҶ…зҪ®дёӨдёӘдҪҺеҠҹиҖ— XtensaВ® 32-bit LX6 MCUгҖӮзүҮдёҠеӯҳеӮЁеҢ…жӢ¬пјҡ

# 4.2 еӨ–йғЁ Flash е’Ң SRAM

ESP32 ж”ҜжҢҒеӨҡдёӘеӨ–йғЁ QSPI flash е’ҢйқҷжҖҒйҡҸжңәеӯҳеӮЁеҷЁ (SRAM)гҖӮиҜҰжғ…еҸҜеҸӮиҖғгҖҠESP32 жҠҖжңҜеҸӮиҖғжүӢеҶҢгҖӢ дёӯзҡ„ SPI з« иҠӮгҖӮESP32 иҝҳж”ҜжҢҒеҹәдәҺ AES зҡ„зЎ¬д»¶еҠ и§ЈеҜҶеҠҹиғҪпјҢд»ҺиҖҢдҝқжҠӨејҖеҸ‘иҖ… flash дёӯзҡ„зЁӢеәҸе’Ңж•°жҚ®гҖӮ

ESP32 еҸҜйҖҡиҝҮй«ҳйҖҹзј“еӯҳи®ҝй—®еӨ–йғЁ QSPI flash е’Ң SRAMпјҡ

вҖў еӨ–йғЁ flash еҸҜд»ҘеҗҢж—¶жҳ е°„еҲ° CPU жҢҮд»Өе’ҢеҸӘиҜ»ж•°жҚ®з©әй—ҙгҖӮ

вҖ“ еҪ“жҳ е°„еҲ° CPU жҢҮд»Өз©әй—ҙж—¶пјҢдёҖж¬ЎжңҖеӨҡеҸҜжҳ е°„ 11 MB + 248 KBгҖӮеҰӮжһңдёҖж¬Ўжҳ е°„и¶…иҝҮ 3 MB + 248 KBпјҢеҲҷ cache жҖ§иғҪеҸҜиғҪз”ұдәҺ CPU зҡ„жҺЁжөӢжҖ§иҜ»еҸ–иҖҢйҷҚдҪҺгҖӮвҖ“ еҪ“жҳ е°„еҲ°еҸӘиҜ»ж•°жҚ®з©әй—ҙж—¶пјҢдёҖж¬ЎжңҖеӨҡеҸҜд»Ҙжҳ е°„ 4 MBгҖӮж”ҜжҢҒ 8-bitгҖҒ16-bit е’Ң 32-bit иҜ»еҸ–гҖӮ

вҖў еӨ–йғЁ SRAM еҸҜжҳ е°„еҲ° CPU ж•°жҚ®з©әй—ҙгҖӮдёҖж¬ЎжңҖеӨҡеҸҜжҳ е°„ 4 MBгҖӮж”ҜжҢҒ 8-bitгҖҒ16-bit е’Ң 32-bit и®ҝй—®гҖӮ

жң¬ж–ҮжЎЈеҸҜиғҪеј•з”ЁдәҶз¬¬дёүж–№зҡ„дҝЎжҒҜпјҢжүҖжңүеј•з”Ёзҡ„дҝЎжҒҜеқҮдёәвҖңжҢүзҺ°зҠ¶вҖқжҸҗдҫӣпјҢд№җй‘«дёҚеҜ№дҝЎжҒҜзҡ„еҮҶзЎ®жҖ§гҖҒзңҹе®һжҖ§еҒҡд»»дҪ•дҝқиҜҒгҖӮ

д№җй‘«дёҚеҜ№жң¬ж–ҮжЎЈзҡ„еҶ…е®№еҒҡд»»дҪ•дҝқиҜҒпјҢеҢ…жӢ¬еҶ…е®№зҡ„йҖӮй”ҖжҖ§гҖҒжҳҜеҗҰйҖӮз”ЁдәҺзү№е®ҡз”ЁйҖ”пјҢд№ҹдёҚжҸҗдҫӣд»»дҪ•е…¶д»–д№җй‘«жҸҗжЎҲгҖҒи§„ж јд№ҰжҲ–ж ·е“ҒеңЁд»–еӨ„жҸҗеҲ°зҡ„д»»дҪ•дҝқиҜҒгҖӮ

д№җй‘«дёҚеҜ№жң¬ж–ҮжЎЈжҳҜеҗҰдҫөзҠҜз¬¬дёүж–№жқғеҲ©еҒҡд»»дҪ•дҝқиҜҒпјҢд№ҹдёҚеҜ№дҪҝз”Ёжң¬ж–ҮжЎЈеҶ…дҝЎжҒҜеҜјиҮҙзҡ„д»»дҪ•дҫөзҠҜзҹҘиҜҶдә§жқғзҡ„иЎҢдёәиҙҹиҙЈгҖӮжң¬ж–ҮжЎЈеңЁжӯӨжңӘд»ҘзҰҒжӯўеҸҚиЁҖжҲ–е…¶д»–ж–№ејҸжҺҲдәҲд»»дҪ•зҹҘиҜҶдә§жқғи®ёеҸҜпјҢдёҚз®ЎжҳҜжҳҺзӨәи®ёеҸҜиҝҳжҳҜжҡ—зӨәи®ёеҸҜгҖӮ

Wi-Fi иҒ”зӣҹжҲҗе‘ҳж Үеҝ—еҪ’ Wi-Fi иҒ”зӣҹжүҖжңүгҖӮи“қзүҷж Үеҝ—жҳҜ Bluetooth SIG зҡ„жіЁеҶҢе•Ҷж ҮгҖӮ

ж–ҮжЎЈдёӯжҸҗеҲ°зҡ„жүҖжңүе•Ҷж ҮеҗҚз§°гҖҒе•Ҷж Үе’ҢжіЁеҶҢе•Ҷж ҮеқҮеұһе…¶еҗ„иҮӘжүҖжңүиҖ…зҡ„иҙўдә§пјҢзү№жӯӨеЈ°жҳҺгҖӮ
