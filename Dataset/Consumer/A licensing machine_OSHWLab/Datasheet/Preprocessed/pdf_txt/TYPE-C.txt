Component ID: USB6
Component Type: USB Type-C Connector
Description: The USB6 is a 16-pin USB Type-C female receptacle designed for power delivery and USB 2.0 data communication. This specific configuration supports reversible plug orientation and is commonly used in devices where USB 3.0 or 3.1 high-speed data pairs are not required. The symmetric pin layout ensures that power and basic data lines are connected regardless of the cable orientation.

Key Parameters:
- Connector Type USB Type-C Receptacle
- Number of Pins 16 Pins
- Current Rating 3.0A to 5.0A
- Voltage Rating 20.0V DC
- Data Protocol USB 2.0 High Speed (480 Mbps)
- Mating Durability 10000 Cycles
- Operating Temperature -30 to +85 Celsius
