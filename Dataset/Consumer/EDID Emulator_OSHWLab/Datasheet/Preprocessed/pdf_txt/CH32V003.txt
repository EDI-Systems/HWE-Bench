Component ID: CH32V003
Component Type: 32-bit RISC-V Microcontroller (MCU)
Description:
The CH32V003 is an industrial-grade general-purpose microcontroller based on the Qingke RISC-V2A core (RV32EC ISA). It operates at up to 48 MHz, supports wide-voltage operation (2.7V to 5.5V), and features low-power modes, a single-wire debug interface, and small package options. Key peripherals include 16KB of Code Flash, 2KB of SRAM, a 10-bit ADC with up to 8 external channels, one operational amplifier/comparator, multiple timers (including an advanced timer for motor control), and standard communication interfaces (USART, I2C, SPI). It is designed for cost-sensitive embedded applications requiring efficient processing and connectivity.

Key Parameters:
- Core: 32-bit Qingke RISC-V (RV32EC instruction set)
- Max Clock Speed: 48 MHz
- Program Memory (Flash): 16 KB
- Data Memory (SRAM): 2 KB
- Operating Voltage (VDD): 2.7 V to 5.5 V (3.3V or 5V nominal)
- Operating Temperature: -40°C to 85°C
- ADC Resolution: 10-bit
- GPIO Count: Up to 18 (varies by package)
- Low-Power Modes: Sleep, Standby
- Debug Interface: Serial Single-Wire Debug
