Component ID: CH340
Component Type: USB to Serial/UART Bridge IC
Description:
The CH340 is a USB bus adapter chip that converts USB to a standard asynchronous serial (UART) interface, enabling communication between a computer and serial devices. It emulates a full-featured COM port with hardware flow control signals (RTS, DTR, DCD, RI, DSR, CTS), supports a wide range of baud rates, and can interface with RS232, RS485, or RS422 through external level shifters. It includes built-in firmware, is compatible with standard USB V2.0 full-speed, and several variants feature an internal clock, eliminating the need for an external crystal.

Key Parameters:
- USB Specification: USB 2.0 Full-Speed
- Supply Voltage: 5V or 3.3V
- Baud Rate Range: 50 bps to 2 Mbps
- Serial Interface: Hardware Full-Duplex UART
- Flow Control Signals: Supports RTS, DTR, DCD, RI, DSR, CTS
- Infrared Support (CH340R): IrDA SIR, 2400 bps to 115200 bps
- Internal Clock: Available on CH340C/N/K/E/X/B variants
- Package Options: SOP-16, SOP-8, SSOP-20, ESSOP-10, MSOP-10
