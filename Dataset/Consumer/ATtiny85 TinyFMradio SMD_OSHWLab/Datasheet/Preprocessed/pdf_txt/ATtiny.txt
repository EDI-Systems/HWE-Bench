Component ID: ATtiny85
Component Type: 8-bit AVR Microcontroller
Description:
The ATtiny25/45/85 is a family of low-power, high-performance 8-bit AVR microcontrollers based on a RISC architecture, featuring in-system programmable Flash memory for program storage. It integrates a core with 32 general-purpose registers, SRAM, EEPROM, multiple timers with PWM, a USI for serial communication, a 10-bit ADC, an analog comparator, and a watchdog timer. Designed for embedded control applications, it offers various power-saving modes and operates across a wide voltage range, with the number of I/O lines and memory size varying by specific model (2/4/8KB Flash, 128/256/512B EEPROM/SRAM).

Key Parameters:
- Core Architecture: 8-bit AVR RISC
- CPU Speed: Up to 20 MHz (model and voltage dependent)
- Program Memory (Flash): 2KB (ATtiny25), 4KB (ATtiny45), 8KB (ATtiny85)
- Data Memory (SRAM): 128B (ATtiny25), 256B (ATtiny45), 512B (ATtiny85)
- EEPROM: 128B (ATtiny25), 256B (ATtiny45), 512B (ATtiny85)
- Operating Voltage (VCC): 1.8V - 5.5V (V-series), 2.7V - 5.5V (standard)
- General Purpose I/O Lines: 6 (Port B)
- ADC Resolution & Channels: 10-bit, 4 single-ended channels
- Package Options: 8-pin PDIP, 8-pin SOIC, 20-pad QFN/MLF, 8-pin TSSOP (ATtiny45/V only)
