# General Description

The TPRT9013 is a high-performance, 500mA LDO regulator, offering extremely high PSRR and ultra-low dropout. Ideal for portable RF and wireless applications with demanding performance and space requirements.

The TPRT9013 is available in 5pin SOT23-5 Package.1 output standards of 1.2V1.5V1.8V2.5V2.8V 3.0V3.3V

# Features

Quiescent Current:60uA   
• PSRR:75dB@1KHz <1uA current at shutdown mode   
• Output voltage accuracy: tolerance ±2%   
• Output current:500mA(Typ.) SOT23-5 package

# Applications

CDM/GSM mobile phone PDAs/MP3 AudioVideo equipment
