Component ID: CP2102
Component Type: USB-to-UART Bridge Controller
Description: The CP2102N-A02-GQFN28R is a high-performance, small-form-factor USB-to-UART bridge from Silicon Labs' USBXpress family. It is designed to provide a simple solution for updating RS-232 designs to USB using minimum components and PCB space. This device includes a USB 2.0 full-speed function controller, USB transceiver, oscillator, and a universal asynchronous receiver/transmitter (UART) in a single package. It is specifically updated (A02 version) to include features like battery charger detection and remote wakeup.

Key Parameters:
- USB Specification: USB 2.0 Full Speed (12 Mbps)
- UART Baud Rate: Up to 3 Mbps
- Integrated Regulator: 3.3V output (supports 3.0V to 5.25V input)
- Package: 28-pin QFN (5x5 mm)
