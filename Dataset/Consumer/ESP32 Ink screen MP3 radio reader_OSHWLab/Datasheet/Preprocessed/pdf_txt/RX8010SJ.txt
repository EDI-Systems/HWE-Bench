Component ID: RX8010SJ
Component Type: Real-Time Clock (RTC) Module with Integrated Crystal and I²C Interface
Description:
The RX8010SJ is a low-power real-time clock module that integrates a frequency-adjusted 32.768 kHz crystal unit. It provides a full calendar (year, month, day, weekday, hour, minute, second), alarm, timer, and frequency output functions. Communication is handled via a 400 kHz I²C interface. It features ultra-low backup current for battery-powered applications and includes 128 bits of user SRAM.

Key Parameters:
- Operating Voltage Range (VDD): 1.6 V to 5.5 V
- Timekeeping Voltage Range (VCLK): 1.1 V to 5.5 V
- Backup Current (Typical @ 3V): 160 nA
- I²C Bus Speed: 400 kHz
- Integrated Crystal Frequency: 32.768 kHz
- Frequency Output Options: 32.768 kHz, 1024 Hz, 1 Hz (CMOS or Open-Drain selectable)
- User RAM: 128 bits (8-bit x 16)
- Operating Temperature Range: -40 °C to +85 °C
