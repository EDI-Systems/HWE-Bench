Component ID: ATtiny
Component Type: Microcontroller (MCU)  
Description: The ATtiny212/214/412/414 Automotive are members of the tinyAVR 1-series, featuring an AVR processor with hardware multiplier operating at up to 16 MHz. They include 2/4 KB Flash, 128/256 bytes SRAM, and 64/128 bytes EEPROM, available in 8-pin and 14-pin SOIC packages. Key peripherals include multiple timers (TCA, TCB, TCD, RTC), a USART, SPI, I2C, analog features (10-bit ADC, 8-bit DAC, Analog Comparator), an Event System, Configurable Custom Logic (CCL), and a Single-Pin Unified Program and Debug Interface (UPDI). Designed for low-power and flexible operation with Core Independent Peripherals, they support automotive temperature ranges and multiple clock options.

Key Parameters:
- Architecture: 8-bit AVR RISC (up to 20 MHz)
- Memory: 4 KB Flash / 256 B SRAM / 128 B EEPROM
- Voltage Range: 1.8V to 5.5V
- Peripherals: 12-bit ADC, 8-bit DAC, USART, SPI, I2C, Event System
