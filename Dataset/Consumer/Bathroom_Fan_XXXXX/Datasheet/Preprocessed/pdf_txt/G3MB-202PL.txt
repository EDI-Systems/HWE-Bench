Component ID: G3MB
Component Type: Solid State Relay (SSR), Phototriac Output, Single In-line Package (SIP)
Description: This is a low-cost, subminiature PCB-mounting solid-state relay in a SIP package designed to switch AC loads up to 2A at 100-240VAC. It features optical isolation between input and output, with variants offering zero-cross switching (G3MB-202P) or non-zero-cross switching (G3MB-202PL). Key integrated features include a built-in snubber circuit and, on most models, a built-in current-limiting input resistor for specified DC voltages (5, 12, 24 VDC). Models without the input resistor (suffix "EG") require an external series resistor to limit the LED current to the recommended 7-20 mA range.
