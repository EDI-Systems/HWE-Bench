Component ID: KX134ACR-LBZE2
Component Type: High Performance Tri-axis Accelerometer
Description: The KX134ACR-LBZE2 is a high-performance, low-power, tri-axis accelerometer with a digital I2C/SPI interface. It is designed for high-g applications and features user-selectable full-scale ranges of +/-8g, +/-16g, +/-32g, and +/-64g. It includes an integrated 512-byte FIFO buffer, advanced power management, and internal digital filters to improve signal quality and reduce system power consumption.

Key Parameters:
- Full-scale Range User selectable +/-8g, +/-16g, +/-32g, +/-64g
- Supply Voltage 1.7V to 3.6V
- I/O Voltage 1.2V to 3.6V
- Interface I2C (up to 3.4MHz) and SPI (up to 10MHz)
- Sensitivity 0.061 to 0.488 mg/LSB
- Output Data Rate 0.781Hz to 25600Hz
- Operating Temperature -40 to +105 Celsius
- Package LGA-12
