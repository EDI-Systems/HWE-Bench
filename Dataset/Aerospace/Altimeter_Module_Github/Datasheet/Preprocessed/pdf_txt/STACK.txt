Component ID: Stack
Component Type: Male PCB Edge Connector
Description: This is a high-density male PCB edge connector (gold finger style) designed for stackable modular systems. It provides standardized power delivery and communication paths between multiple PCB layers. The pinout includes dedicated lines for CAN bus communication and redundant power rails (3.3V, 5.0V, and Battery voltage) paired with ground references. The "Reserved" pins allow for future expansion or proprietary signal routing within the stack.

Key Parameters:
- Connector Style Male Edge Connector
- Interface Type CAN Bus and Power Distribution
- Redundancy Dual Power/GND Groups (A and B)
- Mounting PCB Gold Finger / Edge Mount
- Operating Voltage 3.3V to Battery Voltage
- Operating Temperature -40 to +105 Celsius
