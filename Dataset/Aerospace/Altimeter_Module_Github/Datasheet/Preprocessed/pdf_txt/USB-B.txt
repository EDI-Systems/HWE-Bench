Component ID: USB
Component Type: USB Type-C Female Receptacle (USB 2.0 Only)
Description: The J4 is a 16-pin female USB Type-C connector designed for power delivery and USB 2.0 data communication. It features a simplified pinout that excludes high-speed SuperSpeed differential pairs, making it an efficient choice for devices relying on 480 Mbps data rates. The design includes Configuration Channel (CC) pins for cable orientation detection and power negotiation, as well as Sideband Use (SBU) pins for secondary signaling. The integrated shield pins provide EMI protection and mechanical stability.

Key Parameters:
- Connector Type USB Type-C Receptacle
- Number of Pins 16 Pins
- Data Protocol USB 2.0 High Speed (480 Mbps)
- Maximum Voltage 20V DC
- Current Rating 3A to 5A
- Durability 10000 mating cycles
- Operating Temperature -30 to +85 Celsius
