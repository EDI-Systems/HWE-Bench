Component ID: MPU-6050
Component Type: Six-Axis (Gyro + Accelerometer) MEMS MotionTracking Device
Description: The MPU-6050 is a high-performance motion tracking device that integrates a 3-axis gyroscope and a 3-axis accelerometer on a single silicon die. It features an onboard Digital Motion Processor (DMP) capable of offloading complex MotionFusion algorithms from the host processor. The device includes a primary I2C interface for communication with the MCU and an auxiliary I2C bus to interface with external sensors such as magnetometers to provide complete 9-axis output.

Key Parameters:
- Gyroscope Range +/- 250, 500, 1000, 2000 deg/s
- Accelerometer Range +/- 2, 4, 8, 16 g
- Supply Voltage VDD 2.375V to 3.46V
- Logic Voltage VLOGIC 1.71V to VDD
- Communication I2C up to 400kHz
- Integrated 16-bit ADCs for each axis
- Operating Temperature -40 to +85 Celsius
- Package 24-pin QFN
