Component ID: U6 STM32F103CBT6
Component Type: 32-bit ARM Cortex-M3 Microcontroller
Description: The STM32F103CBT6 is a medium-density performance line microcontroller incorporating the high-performance ARM Cortex-M3 32-bit RISC core operating at a 72 MHz frequency. It features 128KB of Flash memory, 20KB of SRAM, and an extensive range of enhanced I/Os and peripherals. These include two 12-bit ADCs, three general-purpose 16-bit timers plus one PWM timer, as well as standard and advanced communication interfaces including I2C, SPI, USART, CAN, and USB. It is designed for applications such as industrial control, motor drives, and consumer electronics.

Key Parameters:
- Core ARM 32-bit Cortex-M3
- Max CPU Frequency 72 MHz
- Flash Memory 128 KB
- SRAM 20 KB
- Operating Voltage 2.0V to 3.6V
- ADC 2 x 12-bit (10 channels)
- I/O Pins 37
- Operating Temperature -40 to +85 Celsius
- Package LQFP-48
