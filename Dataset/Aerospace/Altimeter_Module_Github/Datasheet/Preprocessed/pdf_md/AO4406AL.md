# Description

The AO4406AL uses advanced trench technology to provide excellent RDS(ON), low gate charge and operation with gate voltages as low as 4.5V. This device is suitable for use as a Battery protection or in other Switching application.

# General Features

VDS = 30V ID =15A RDS(ON) < 10mΩ @ VGS=10V

# Application

Battery protection   
Load switch   
Uninterruptible power supply
