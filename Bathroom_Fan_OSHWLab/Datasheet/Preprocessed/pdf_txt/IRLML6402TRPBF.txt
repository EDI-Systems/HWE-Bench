Component ID: IRL
Component Type: P-Channel Enhancement Mode MOSFET
Description: This is a P-Channel MOSFET utilizing advanced trench technology, designed for efficient power and load switching. It features a low on-state resistance (Rds(on)) down to 39mΩ and low gate charge, enabling fast switching with minimal drive power. It is suitable for logic-level interfacing and power management applications.

| Pin Number | Pin Name | Function | Description | Pin Type | Electrical Type |
| ----- | -------- | -------- | ----------- | -------- | --------------- |
| 1 | G | Gate | Control terminal for turning the MOSFET on/off. Voltage applied relative to the Source pin. | Input | Analog, High Impedance |
| 2 | D | Drain | Output/load terminal. Current flows from Source to Drain when the MOSFET is on. | Power Output | Power, Low Resistance when ON |
| 3 | S | Source | Input/power terminal. Common reference point for Gate voltage and current return path. | Power Input | Power, Common Reference |
