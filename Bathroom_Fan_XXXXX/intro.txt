Please design a circuit that must include the following two modules:

The first is a power switching and self-latching module with button state detection. This module should utilize a P-channel MOSFET as the primary power switch and an N-channel MOSFET to implement the logic latching. The circuit needs to support an initial power-on trigger via a low-level signal from a physical button; once the logic control unit starts, it must output a high-level signal to drive the N-channel MOSFET, maintaining the power path and ensuring the system remains powered after the button is released. Additionally, a diode network should be used for signal isolation and logical decoupling, ensuring the button can still be independently detected by the control unit as an input signal while the power is self-latched.

The second is an isolated driver control module with safety protection and status indication. This module should use a switching transistor as the driving core, receiving a logic high-level signal to control a Solid State Relay (SSR). To ensure system stability, a pull-down resistor must be configured at the driver input to prevent any misoperation when the logic unit is initializing or in a floating state. Furthermore, a Light Emitting Diode (LED) should be integrated into the control loop to provide synchronized visual feedback whenever the relay is activated.”
